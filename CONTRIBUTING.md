# Contributing to PBody FullStack Academy

We welcome contributions! Here's how to get started:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/pbody-fullstack-academy.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Push to your branch: `git push origin feature/your-feature`
6. Create a Pull Request

## Development Setup

```bash
# Install dependencies
npm install

# Install frontend
cd frontend && npm install && cd ..

# Install backend
cd backend && npm install && cd ..

# Start development
npm run dev
```

## Code Style

- Use ES6+ features
- Follow the existing code structure
- Add comments for complex logic
- Test your changes

## Commit Messages

- Use clear, descriptive messages
- Start with an emoji: 🎨 (style), 🐛 (bug fix), ✨ (feature), 📚 (docs)
- Example: "✨ Add new lesson on React hooks"

## Pull Request Process

1. Update README.md with any new features
2. Add tests if applicable
3. Ensure your code follows the style guide
4. Request review from maintainers
